## Contribute

1. [Fork this repo.](https://github.com/hctilg/recursive_build_sum/fork)
2. Create a new directory in the "Languages" directory with the name of the programming language with lowercase letters.
3. Write some code with the **Recursive Build Sum Algorithm** in that directory.
4. Write a README.md file (like [this file](languages/rust/README.md)) in the same directory and provide instructions on how to run your code.
5. In the main path of the repo, in the same [`README.md`](README.md) file in the [languages ​​section](#languages), add the directory address with the name of the programming language.
6. [Send a clean pull request.](https://github.com/hctilg/recursive_build_sum/compare)
7. Hug yourself from us :)
